### 🏗 Flagship: FY26JaxWins — AI-Powered Job Search Operating System

> Jax is the product. Target companies are the accounts. This is a full personal GTM motion, not a job hunt.

A CLAUDE.md-governed Obsidian vault that runs the entire job search like a B2B sales campaign. Root-level AI governance, persistent memory system, 3 specialized agents, 7 skill-based slash commands, and per-company deal rooms under `targets/[company]/`. Every output (research, decks, outreach, interview prep, resume) is grounded in identity files stored in `_context/`. One command triggers the whole pipeline.

`Claude Code` `Claude API` `Python` `PptxGenJS` `Apollo MCP` `n8n` `Obsidian` `Airtable`

### The Agent Trio

> Every agent is named after a character who embodies what it does. These aren't random names. They're philosophies turned into code.

<table>
<tr>
<td width="50%">

### Agent Mahomes — Company Research Agent
**Full-Field Intelligence Before the Snap**

*Inspired by Alexander Mahone from Prison Break — the only person smart enough to catch Michael Scofield. Mahone studied patterns, mapped behaviors, and saw what nobody else could see. This agent does the same thing to companies.*

Give it a company name and domain. It runs a multi-phase intel pipeline: Apollo enrichment, leadership mapping, web research, brand color extraction from CSS, 8-point red flag scan. Delivers a branded Word dossier + POV brief with a recommended narrative angle. The same approach Jax used to book enterprise deals at UpKeep — study the target like Mahone studied Scofield.

`Claude Code` `Claude API` `Apollo MCP` `Python` `BeautifulSoup`

</td>
<td width="50%">

### Agent Draper — Branded GTM Deck Builder
**Narrative Arc to PPTX in One Command**

*Inspired by Don Draper from Mad Men — the man who turned a Kodak slide projector into "The Carousel" and made a room full of executives feel something. Every great deal starts with a story told so well they forget you're selling. That's why this agent exists.*

Takes a company research file and identity context, defines a narrative arc, writes all deck content, then generates a full 9-slide PPTX in the company's actual brand colors (pulled from brand.json). Slides 1-7 in PptxGenJS, Slides 8-9 appended via python-pptx. Includes speaker notes script and a 3-slide executive punch version. Standard mode + assessment mode for take-home case studies.

`Claude Code` `PptxGenJS` `python-pptx` `Node.js` `Python`

</td>
</tr>
<tr>
<td width="50%">

### Agent Scofield — People Intelligence Agent
**Map the Room Before You Walk In**

*Inspired by Michael Scofield from Prison Break — the man who tattooed the entire blueprint on his body. Every person, every connection, every escape route mapped before he ever stepped inside. This agent does the same thing to people.*

Three modes: room (pre-interview panel mapping), deal (active deal stakeholder intel), and prospect (org chart for cold outreach). Takes names or emails, enriches via Apollo, analyzes career patterns and reporting structure. Delivers an interactive HTML link chart with draggable nodes + a written strategy brief with conversation angles for each person.

`Claude Code` `Claude API` `Apollo MCP` `JavaScript` `HTML/CSS`

</td>
<td width="50%">

### Build Resume Agent — JD Scoring + Tailoring
**Never Apply to a Bad-Fit Role Again**

Takes a JD and company name, runs a weighted scoring model against the master resume to calculate a fit score, selects the correct variant (AE, Founding AE, or GTM Engineer), tailors bullets using proof points, and saves a JD scorecard + ready-to-send resume. Prevents wasted applications and ensures every resume is grounded in real numbers.

`Claude Code` `Claude API` `Markdown`

</td>
